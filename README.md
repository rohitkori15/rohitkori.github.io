## 📌 About the Project

This portfolio is designed to present my:
- Educational background
- Professional experience
- Technical and analytical skills
- Certifications and achievements
- Personal interests and languages

It uses **HTML, CSS, and JavaScript** with smooth scrolling and animations to create an engaging user experience.

---

## 🧩 Features

- 📌 Fixed navigation bar with smooth scrolling  
- 🎯 Full-screen hero section  
- 📚 Timeline-style education section  
- 💼 Alternating layout for experience section  
- 🎴 Card-based UI for skills, certifications, and interests  
- ✨ Scroll-based fade-in animations  
- 📱 Responsive grid design  

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3 (Flexbox & Grid)**
- **JavaScript (Intersection Observer API)**
